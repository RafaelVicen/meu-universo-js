📢 Matéria: Comunicação no JavaScript
Existem duas formas principais de o JavaScript mostrar informações: no "esconderijo" dos programadores (Console) e na "cara" do usuário (Pop-ups).

1. O Console (O Diário do Programador)
O console é onde o robô (computador) conta o que está acontecendo por trás das câmeras. Ele tem "tons de voz" diferentes:

console.log() (A fala normal): É o mais comum. Serve para mostrar qualquer mensagem comum.

console.info() (O lembrete): É parecido com o log, mas serve para dar uma informação útil. Em alguns navegadores, ele ganha um ícone pequeno de "i".

console.warn() (O aviso): Pinta a mensagem de amarelo. É como um sinal de trânsito dizendo: "Cuidado, algo pode dar errado!".

console.error() (O erro): Pinta a mensagem de vermelho. É usado quando algo quebrou de verdade.

2. Caixas de Mensagem (Interação com o Usuário)
Essas são aquelas janelinhas que "pulam" na tela do navegador.

alert() (O Grito): Mostra uma mensagem e um botão de "OK". O usuário não consegue fazer nada no site até clicar no OK. É como um sinal de "PARE e LEIA".

confirm() (A Pergunta): Mostra uma mensagem com dois botões: OK ou Cancelar.

Se clicar em OK, ele entende como true (verdadeiro).

Se clicar em Cancelar, ele entende como false (falso).

prompt() (A Conversa): É o mais interativo. Ele mostra a mensagem e abre um espacinho para o usuário escrever alguma coisa (como o nome ou a idade).
